Hoje foi dia de tirar três coisas do papel: devlog automatizado, agente jornalista, e o site que você está lendo agora.

## Devlog automatizado

Skill custom no Claude Code que lê sessões da pasta `.claude/projects/`, faz git log de repos locais e da Larissa via SSH, lê exports do Claude.ai descompactados de uma drop folder, consolida tudo em um devlog do dia, comita e pusha pro repo `devlog-elquer` no GitHub. Trabalho manual recorrente: 1 minuto por dia.

## Agente jornalista

GitHub Action que dispara via schedule diário. Usa `anthropics/claude-code-action@v1` autenticada com OAuth da Max. Lê o devlog do dia, lê instruções de voz/formato em `.claude/posts-prompt.md`, e gera 4 peças adaptadas: blog, YouTube, Instagram, LinkedIn.

Tive que pivotar de `push` pra `schedule + workflow_dispatch` porque a action não suporta evento push. Levou 15s de erro pra descobrir.

## Site

Astro + Cloudflare Pages. Domínio elquer.com apontado, redirects 301 dos secundários configurados via Cloudflare Redirect Rules.

Identidade visual "One-Man Show" aplicada: palco escuro, chão creme, holofote âmbar. Fraunces nos títulos, Inter no corpo, JetBrains Mono pra código.

## Travas do dia

- Cloudflare Pages está sendo absorvido por Workers — interface empurra o usuário pra criar Worker em vez de Pages.
- MailerLite embed form ignora CSS externo. Solução: estilizar dentro do próprio editor do MailerLite.
- Astro deixou um cache que persistiu CSS antigo escondendo o headline do form. Restart do dev server resolveu.
